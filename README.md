# FE-scaffolding-X
Scaffolding for web, Based Baidu FIS 2.  


## Feature
support Less CSS  
CSS auto prefix  


## How to Use ?
installed Node.js and NPM .
	npm i
	npm start
then, open http://127.0.0.1:3000 in browser.

stop http-server
	ctrl + break

some of command in package.json file, for windows user only.
